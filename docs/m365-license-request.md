# Microsoft 365 License and Software Requests

## Requesting a license
Software and license requests are submitted through the IT service portal with manager approval. Standard staff receive a Microsoft 365 E3 license, which includes Outlook, Teams, OneDrive, SharePoint, and the Office desktop apps. Roles that require advanced security or compliance features receive E5.

## Adding access to a shared resource
To request access to a SharePoint site, shared mailbox, or Teams channel, name the exact resource and attach the data owner's approval. Access is granted by adding the user to the resource's security group, not by direct permissions, so access can be audited and removed cleanly when the user changes roles.

## Reassigning or reclaiming licenses
When an employee leaves or changes roles, their license should be reclaimed or reassigned to control cost. Reclaimed E3/E5 licenses return to the available pool. Mailboxes of departed users are converted to shared mailboxes for retention before the license is removed.

## Third-party software
Requests for third-party applications (for example Adobe Acrobat Pro) require manager approval and a license purchase. Installation is performed through the managed software catalog where possible to keep deployments consistent and patchable.
